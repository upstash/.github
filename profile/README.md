# Upstash Serverless Data Platform

Upstash is a Serverless Data Platform offering Redis, Vector Database, QStash, Workflow, and Box APIs. It’s the perfect solution for your serverless applications, providing:

- Low latency data
- Per request pricing with a monthly cap
- Global replication
- Ease of use
- Built-in REST API

On top of Redis, Upstash also offers extensions like Search, Ratelimit, and Realtime to build richer applications.

Whether you need fast in-memory storage, AI-powered vector search, reliable messaging, durable workflows, or sandboxed compute for AI agents, Upstash has you covered.

## Products

- [Redis](https://upstash.com/docs/redis/overall/getstarted) — Serverless Redis with global replication and per-request pricing.
- [QStash](https://upstash.com/docs/qstash/overall/getstarted) — Serverless messaging and scheduling for the HTTP-based serverless world.
- [Workflow](https://upstash.com/docs/workflow/getstarted) — Durable, serverless workflows that run on top of QStash.
- [Vector](https://upstash.com/docs/vector/overall/getstarted) — Serverless vector database for AI-powered semantic search.
- [Box](https://upstash.com/docs/box/overall/quickstart) — Serverless sandboxes to run code and AI agents securely.

### Redis Extensions

- [Search](https://upstash.com/docs/redis/search/introduction) — Full-text search built on Upstash Redis.
- [Ratelimit](https://upstash.com/docs/redis/sdks/ratelimit-ts/overview) — Connectionless rate limiting for serverless and edge runtimes.
- [Realtime](https://upstash.com/docs/realtime/overall/quickstart) — Realtime pub/sub channels powered by Upstash Redis.
